# Sistema de Analise de Dividas

Este repositorio organiza a base estrategica e tecnica de uma operacao voltada a direito bancario, relacao consumerista e conflitos com instituicoes financeiras.

## Estrutura atual

- Base editorial para eBook e conteudo juridico informativo
- Estrategia comercial e de atendimento
- PRD, arquitetura e guias para desenvolvimento por agentes
- Prototipo estatico e base em Next.js
- Persistencia inicial de leads com caminho preparado para Supabase

## Status

A estrutura local do projeto ja contem landing page, simulador, esteira inicial de leads, painel interno e documentacao de implementacao.
